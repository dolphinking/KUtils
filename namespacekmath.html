float kmath::intervalProject ( float  val,
const Interval &  domain,
const Interval &  range 
)

Project a value within domain to a new interval range using func.

The value is mapped, using domain, onto the function's domain, then the function is called on that value. The result is mapped back, using funcRange, onto the output range.

With the default arguments, the behavior is to project the value linearly from domain to range.

Parameters:
valThe value to map.
domainDomain of val.
rangeRange of output.
funcCalled on the result of mapping val onto funcDomain.
funcDomainDomain of func.
funcRangeRange of func.
boundIf true, val is first bounded to the closest point in domain.
Exceptions:
range_errorThrown if bound is false and val is not within domain, or func returns something outside of funcRange.
