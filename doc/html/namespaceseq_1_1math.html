Return XRange sequence, similar to Python's "xrange".

Suggested usage: 

 for (auto i : xrange(10)) {
     printf("%d\n", i);
 }

Unlike the Python version, this can also accept floats/doubles, and can include the endpoint of the range by specifying inclusive = true.
